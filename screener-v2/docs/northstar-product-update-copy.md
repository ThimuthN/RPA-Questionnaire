# Northstar Update

## Short Intro

Northstar is starting to feel like one product.

Hiring flow, assessment setup, candidate review, and results now connect more clearly.

## Main Points

- New assessments are now more useful for real hiring work
- Strict mode is easier to see during setup
- The candidates table is becoming a real control surface
- The jobs pipeline is taking shape
- Light mode and dark mode both have a clear direction
- Candidate detail now shows more of the real hiring flow
- Resume handling is easier to access
- Results are easier to review
- The in-test experience is more structured

## Close

This is no longer just a rebrand update.

It is a product update.
