# BA IQ Test | 1st Screening

This add-on provides a 30-question BA screening module using a placeholder workflow screenshot. It tests candidates' ability to interpret visual risk signals, clarify process risk, and select the safest next action.

Files:
- `questions.ts` contains the authored BA screening question content and placeholder image prompt blocks.
- `definition.ts` exports the add-on definition and library entry.
- `questions.test.ts` verifies the question set builds as expected.
